# React + Vite

## Project name: KeenKeeper

## Description

KeenKeeper is a simple web app to help you manage and maintain your personal relationships.
It lets you track when you last contacted friends and reminds you to stay connected.


## Technologies Used
React.js
Vite
Tailwind CSS
DaisyUI
React Router

## Key Features
Track last contact time with friends
Filter friends by status (On Track, Almost Due, Overdue)
Organize friends with tags (Work, Family, Hobby, Travel)
